1. DISC1과 DISC2의 내용물을 하나의 폴더에 압축해제
2. 관리자 cmd로 cd (폴더) 들어가서 

copy /b X3._1 + X3._2 + X3._3 X3_.big ( _ = 0/1/2)

이러면 총 X30.big/X31.big/X32.big 3개의 파일이 나온다

3. 명령줄 커맨드 "Xeno23Lbae.exe X3._0 Lba_.txt"로 각 X3._0 파일에서
파일 리스트를 .txt로 복호화함

4. 명령줄 커맨드(파워셸) "Xenounpack.exe Lba_.txt X3_.big"로 각 파일의 내용을
압축 해제하면 UNPACKED 폴더가 생김

5. 폴더 안 내용 씹뜯맛즐 하면 됨

6. 파일 내용 변경 후 
Xenorepack.exe <<< 이 부분은 좀 더 자세히 알아봐야 함(프로그램에서 리소스 인식이나 정확한 방식은 어떻게 하는지 등)
(참고: 리팩할 때는 리팩할 데이터의 폴더명을 'UNPACKED'로 지을 것)
